# Diaa Store

> نظام متكامل لإدارة الاشتراكات والمبيعات والعملاء

## 🚀 المميزات

- **لوحة تحكم** — إحصائيات ورسوم بيانية شاملة
- **إدارة المبيعات** — تسجيل ومتابعة عمليات البيع والتجديد
- **إدارة المنتجات** — إضافة وتعديل المنتجات والأسعار
- **إدارة المخزون** — أكواد التفعيل والحسابات مع نظام مجلدات
- **إدارة العملاء** — قاعدة بيانات العملاء ومعلوماتهم
- **إدارة المحافظ** — متابعة أرصدة المحافظ الإلكترونية
- **التقارير** — تقارير مفصلة عن الأداء والمبيعات
- **الشفتات** — نظام إدارة الورديات
- **المصروفات** — تسجيل ومتابعة المصروفات
- **التنبيهات** — إشعارات التجديد والمستحقات
- **إدارة المستخدمين** — صلاحيات متعددة المستويات (Admin / Moderator)

## 🛠️ التقنيات المستخدمة

| التقنية | الغرض |
|---------|-------|
| React 18 | واجهة المستخدم |
| Vite 5 | أداة البناء والتطوير |
| Supabase | قاعدة البيانات والمصادقة |
| TailwindCSS 3 | التصميم |
| Vercel | الاستضافة |

## ⚡ التشغيل محلياً

```bash
# تثبيت المكتبات
npm install

# تشغيل خادم التطوير
npm run dev
```

## 🔐 متغيرات البيئة

أنشئ ملف `.env` في المجلد الرئيسي:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

## 📦 البناء للإنتاج

```bash
npm run build
```

## 🌐 النشر على Vercel

1. ارفع المشروع على GitHub
2. اربط الـ Repository بـ Vercel
3. أضف متغيرات البيئة في إعدادات Vercel
4. سيتم النشر تلقائياً مع كل push

## 📁 هيكل المشروع

```
diaa-store/
├── index.html              # نقطة الدخول
├── public/                 # ملفات ثابتة
├── src/
│   ├── main.jsx           # نقطة تشغيل React
│   ├── App.jsx            # المكون الرئيسي
│   ├── components/        # مكونات الواجهة
│   ├── context/           # إدارة الحالة (Auth + Data)

```
